# AlphaFold Cookbook

An AI-collaboration cookbook for complete beginners who want to explore protein structure prediction without prior biology or programming experience. Here, “cookbook” means a practical, step-by-step path: complete one small foundation at a time, then use that confidence to go further on your own.

Learners act as the chef. AI acts as the sous-chef. Each short, visual step helps the learner select a verified protein sequence, explore or predict its structure, create a rotating 3D visualization, and write a careful beginner-friendly report.

## Cookbook #01

**Bake a Nobel Prize-Winning AI Structure with ChatGPT: An AlphaFold 3D Protein Visualization Recipe**

By the end of the cookbook, a learner will have:

- a verified FASTA sequence from a trusted biological database;
- an AI-predicted or database-sourced protein structure;
- a rotating 3D protein video;
- a short AI-assisted interpretation report; and
- a checklist separating model predictions from verified scientific evidence.

## Environment variables

Deployment values are configured through Coolify environment variables.

```env
APP_DATA_DIR=/data
OPENAI_API_KEY=
OPENAI_MODEL=
ADMIN_PREFIX=/private-admin-path
TELEGRAM_BOT_TOKEN=
```

## Scientific scope

Protein structures generated by AI are predictions with varying confidence. The cookbook teaches learners to read confidence measures and check protein identifiers, sequences, known functions, and binding-site claims against authoritative sources.

## Status

The FastAPI cookbook site and Telegram long-polling worker are deployable with Docker Compose. Cookbook content, sample FASTA files, and additional visual assets remain under development.

## Telegram bot setup

The `bot` service runs separately from the public web service and does not expose a port. Configure these variables in Coolify, then redeploy:

```env
TELEGRAM_BOT_TOKEN=your-token-in-coolify-only
OPENAI_API_KEY=your-key-in-coolify-only
OPENAI_MODEL=gpt-5.4-mini
```

After deployment, open [@AlphaFoldCookbookBot](https://t.me/AlphaFoldCookbookBot) and send `/start`.
